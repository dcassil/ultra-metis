# Implement baseline comparison engine and quality record generation

## Objective

Build the baseline comparison engine that diffs two AnalysisBaselines for the same tool and produces a QualityRecord with metric deltas, new/resolved findings, and trend direction.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] BaselineComparisonEngine: takes two ParsedToolOutputs (before/after), produces QualityRecord
- [ ] Metric deltas: for each metric, compute before/after/delta/direction (improved/regressed/unchanged)
- [ ] Finding diff: identify new findings (in after but not before), resolved findings (in before but not after)
- [ ] Per-file breakdown: which files improved vs regressed
- [ ] QualityRecord body populated with structured comparison data
- [ ] Unit tests with realistic before/after scenarios
